# Progr3s User Guide

This guide is for everyday use of the app.

## What Progr3s is

Progr3s helps you track:
- one-time goals (for example: finish a homework project)
- repeating weekly routines (for example: gym on Tue/Thu/Fri/Sat)

You can add subtasks, collect proof, share with connections, and review completed history in the calendar.

## Quick Start

1. Create your account on `/signup`.
2. Log in on `/login` using either:
- email + password
- username + password
3. Go to your dashboard (`/dashboard`).
4. Add either:
- `Add Goal` for one-time items
- `Add Routine` for weekly repeating items

## Goals vs Routines

- Goal:
- one-time tracking
- progress does not auto-reset weekly

- Routine:
- meant for repeating weekly habits
- routine progress resets at the start of each new week (Sunday rollover)
- your routine structure stays (title, subtasks, schedule), but weekly progress starts fresh

## How to Use the Dashboard

## 1) Add goal/routine
- Enter a title.
- Pick a status.
- For routines, choose scheduled days.

## 2) Add subtasks
- Open the goal/routine card.
- Add subtasks for the exact steps you want to complete.
- Use `Done` / `Undo` to track each subtask.

## 3) Choose verification
- Open `Schedule & Verification`.
- Enable one or more:
- Photo proof
- Geolocation proof
- Peer approval

## 4) Upload proof
- If photo proof is on, upload an image.
- If geolocation is on, use `Capture Geo`.
- If peer approval is on, share with a connection and wait for approval.

## Connections and Sharing

Use the `Connections` page (`/connections`) to:
- send connection requests by username
- accept/decline incoming requests
- remove existing connections

After connecting, you can share goals/routines with them from the dashboard.

## Calendar and History

- Completed and confirmed activity appears in your calendar timeline.
- You can click marked days to review what was completed.
- Proof images are designed to be lightweight/temporary; completion records and text proof metadata remain for history.

## Settings

From `Settings` (`/settings`) you can:
- change password
- reset account data (keeps account)
- permanently delete account

## Tips

- Keep routine subtasks specific (example: Push, Pull, Legs, Cardio).
- Use connections only with people you trust for verification.
- Use the status filter to focus on what is active right now.
